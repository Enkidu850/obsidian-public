## Synchronisation privé -> public

Dans `langage/obsidian`, pour synchroniser le public avec l'état actuel du privé :
```powershell
npm run sync-public
```
Dans `langage/obsidian`, pour synchroniser et commit/push le public avec l'état actuel du privé :
```powershell
npm run sync-and-push
```

Source commandes dans :
`langage/obsidian/package.json`
